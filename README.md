# WalletMate 💰

### Smart Expense Tracker

Flutter와 Dart를 활용하여 개발한 스마트 가계부 애플리케이션입니다.

사용자의 수입과 지출을 기록하고 분석하여 효율적인 자산 관리를 지원하며, 소비 패턴을 시각적으로 확인할 수 있는 서비스를 제공합니다.

---

# 01. 프로젝트 소개

WalletMate는 사용자의 수입과 지출을 기록하고 분석하여 효율적인 자산 관리를 지원하는 스마트 가계부 애플리케이션입니다.

### 주요 목적

* 💰 수입·지출 기록
* 📊 소비 패턴 분석
* 📅 월별 예산 관리
* 📈 통계 시각화 제공

### 개발 배경

최근 모바일 결제와 온라인 소비가 증가하면서 개인의 소비 내역을 체계적으로 관리하는 중요성이 높아지고 있습니다.

WalletMate는 사용자가 자신의 소비 습관을 쉽게 파악하고 계획적인 소비 생활을 실천할 수 있도록 개발되었습니다.

# 03. UI / UX 설계

WalletMate는 사용자가 쉽고 빠르게 소비 내역을 기록하고 확인할 수 있도록 직관적이고 간결한 사용자 경험을 목표로 설계하였습니다.

### UI 특징

#### 보라색 메인 컬러 사용

* 신뢰감과 안정감 제공
* 금융 애플리케이션 이미지 강화

#### 데이터 시각화 강조

* 그래프를 활용한 소비 패턴 분석
* 직관적인 정보 전달

#### 심플한 화면 구성

* 불필요한 요소 최소화
* 핵심 정보 중심 배치

### UX 설계 원칙

1. 쉬운 사용성
2. 빠른 접근성
3. 높은 가독성
4. 시각적 분석 제공

# 04. 화면 구성

### 🏠 Home

총 자산, 이번 달 수입·지출, 예산 사용 현황을 한눈에 확인할 수 있는 메인 화면

### ➕ 거래 추가

수입 및 지출 내역을 등록하고 카테고리, 금액, 메모 등을 입력하는 화면

### 📅 Calendar

날짜별 거래 내역을 확인하고 특정 날짜의 소비 기록을 조회할 수 있는 화면

### 📊 Statistics

카테고리별 소비 비율, 월별 지출 추이 등 소비 패턴을 그래프로 분석하는 화면

### 🧾 Receipt OCR

영수증 사진을 촬영하거나 등록하여 거래 정보를 관리하는 화면

# 05. 기대 효과

### 사용자 측면

#### 효율적인 자산 관리

수입과 지출을 체계적으로 관리하여 자산 현황을 쉽게 파악할 수 있습니다.

#### 소비 습관 개선

소비 패턴 분석을 통해 불필요한 지출을 줄일 수 있습니다.

#### 계획적인 예산 관리

예산 설정 기능을 통해 계획적인 소비 습관을 형성할 수 있습니다.

### 서비스 측면

#### 편리한 기록 관리

영수증 등록 및 통계 기능을 활용하여 보다 편리한 가계 관리가 가능합니다.

# 06. 향후 개선 및 느낀 점

## 향후 개선 계획

### OCR 영수증 인식 기능

* 금액 자동 추출
* 날짜 자동 인식
* 상호명 자동 입력

### AI 소비 분석 기능

* 소비 패턴 분석
* 예산 초과 가능성 예측
* 맞춤형 소비 리포트 제공

### 금융 서비스 연동

* 은행 계좌 연동
* 카드 사용 내역 연동
* 자동 가계부 작성 기능

## 개발하면서 느낀 점

개발 과정에서 단순히 기능을 구현하는 것보다 사용자의 입장에서 서비스를 바라보는 것이 중요하다는 점을 느꼈습니다.

"이 기능이 있으면 더 편리하지 않을까?", "사용자가 이 화면을 사용할 때 불편하지 않을까?"와 같은 고민을 지속적으로 하게 되었고, 이를 통해 사용자 경험(UX)과 사용자 인터페이스(UI)의 중요성을 다시 한번 깨달을 수 있었습니다.

좋은 앱은 기능이 많은 앱이 아니라 사용자가 쉽고 편리하게 사용할 수 있는 앱이라는 점을 배울 수 있었습니다.
